UnityGameは、Unityエンジンを用いて開発された3DRPGゲームです。プレイヤーはキューブとなり、なるべく多くの敵を倒し生き残ることを目指します。本プロジェクトは、3Dゲームの基礎の実装に重点を置いた実験的かつ実践的なアプリケーションです。

## ✨ 主な機能

- **プレイヤーの基礎的なアクション**: 前後左右移動やジャンプ、攻撃など3Dゲームに必要な操作が可能
- **敵の追尾機能**: プレイヤーが範囲内に入れば自動追尾する敵キャラクター
- **視覚的に理解できるHPバー**: DOTweenを使用した本格的なHPバー
- **無限に湧き続ける敵スポナー**: 一定時間ごとに湧き続ける敵キャラクター。バランス調整できるように敵の上限数設定可能


## 🏗️ アーキテクチャ

```
Assets/
├── Scripts/
│   
├── CubeController.cs  # キャラクターの移動・状態管理の中核
├── Weapon.cs          # 武器の当たり判定と攻撃処理
├── Enemy.cs           # 敵の基本ステータス・被ダメージ処理
├── EnemyDetection.cs  # プレイヤー検知（索敵）ロジック
├── EnemySpawner.cs    # 敵の生成・管理システム
├── HealthGauge.cs     # プレイヤーHPバー制御
├── EnemyHealthGauge.cs # 敵HPバー制御
│ 
├── Prefabs/                # キャラクター・HPバーのUI
├── Materials/                # キャラクターの素材
├── Scenes/                 # ゲームシーン管理
└── Animations/              # アニメーションコントローラー
```

## ⚙️ 必要な環境

- Unity 6 (6000.0.21f1) 
- Visual Studio 2022 

## 🚀 セットアップ方法

1. **リポジトリのクローン**
    ```bash
    git clone <https://github.com/kmtkzy1379/MyUnityProject.git>
    ```

2. **プロジェクトクローン追加**
Unity Hubを開き、「Add (追加)」ボタンからクローンしたフォルダを選択してください。

3. **エディタの起動**
エディタの起動プロジェクトをクリックし、Unity Editorを起動します。（初回起動時はライブラリのインポートに時間がかかる場合があります）

4. **ゲームの実行**
ゲームの実行エディタ上部の再生（▶）ボタンを押してゲームを開始してください。

## 📝 使用方法

- ↑…上方向移動
- →…右方向移動
- ↓…下方向移動
- ←…左方向移動
- shift…ダッシュ
- space…ジャンプ
- A…左方向カメラ回転
- W…右方向カメラ回転
- C…攻撃
